Mapbox

Mapbox is a jQuery plugin which shows a map in a modal dialog box, much like the well known Lightbox jQuery plugin.

Example

Click on the link below to see an example:

Show map

The following snippets were used to create this link.

HTML: 

<a class="mapbox" href="https://maps.google.com/maps?f=q&source=s_q&hl=en&geocode=&q=san+diego,+ca&aq=&sll=37.0625,-95.677068&sspn=56.899383,104.941406&ie=UTF8&hq=&hnear=San+Diego,+California&t=m&z=11&ll=32.715329,-117.157255&output=embed">Show map</a>

JavaScript: 

$(document).ready(function() {
	$(".mapbox").mapbox();
});

Of course, you can use any CSS class name or indeed any jQuery selector you like to target links for Mapbox-ification.
