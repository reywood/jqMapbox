jqMapbox

jqMapbox is a jQuery plugin which shows a map in a modal dialog box, much like the well known Lightbox jQuery plugin.

JavaScript: 

$(document).ready(function() {
	$(".jqmapbox").jqmapbox();
});

Of course, you can use any CSS class name or indeed any jQuery selector you like to target links for jqMapbox-ification.

Configuration

The jqMapbox plugin has some configurable options you can use to control the look of the modal dialog. These options are passed to the jqmapbox function as a normal JSON object. For example: 

$(".jqmapbox").jqmapbox({
	overlayColor: "#000",
	mapWidth: 600,
	mapHeight: 400
});

Available configuration options are:

mapWidth
The width of the map shown in the dialog. This should be any reasonable integer value. Note that this value controls the width of the actual map, not the width of the dialog. Defaults to 800.
mapHeight
The height of the map shown in the dialog. This should be any reasonable integer value. Note that this value controls the height of the actual map, not the height of the dialog. Defaults to 600.
overlayColor
The color of the overlay shown behind the dialog. This should be a quoted string representing any valid CSS color value (e.g. "#888", "#000000", "rgb(128, 128, 128)"). Defaults to black.
overlayOpacity
The opacity of the overlay shown behind the dialog. This should be a floating point number between 0.0 and 1.0. Defaults to 0.8.
imageBtnClose
The URL of the close button image. This should be a quoted string containing a valid URL pointing to the location of the image. This image shows up in the lower right corner of the dialog which the user can click to close the map. Defaults to "images/jqmapbox-btn-close.gif".
resizeSpeed
The duration of the resize animation when the dialog is opening. This should be an integer value representing the number of milliseconds for the duration. Defaults to 400.
keyToClose
A keyboard key that, when pressed, closes the dialog. This should be a quoted, one character string. Defaults to "x". Note that the ESC key will also close the dialog, regardless of this setting.

jqMapbox also supports an HTML attribute, which can be added to the link tag.

data-jqmapbox-caption
The value of this attribute will be used as the caption for the dialog. The caption will appear underneath the map.
